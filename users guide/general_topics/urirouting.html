URI Routing

این کتابخانه به شما اجازه میدهد تا به مقادیر ارسال شده از طریق Url دسترسی شده باشید. این کتابخانه جزو کتابخانه های پیشفرض سیستم میباشد.

قبل از مطالعه این بخش لازم است بخش EasyDev URLs مطالعه کرده باشید.

 

 

نحوه استفاده از این کتابخانه به شکل زیر میباشد

$this->router

 

حهت دسترسی به اسم فولدری ای که از طریق ادرس Url ارسال شده است طبق مقال زیر عمل کنید:

 $this->router->getDirectory();

خروجی مثال بالا به صورت رشته میباشد.

 

جهت دسترسی به اسم Controller ای که از طریق آدرس Url ارسال شده است طبق مثال زیر عمل کنید:

 $this->router->getClass();

خروجی مثال بالا به صورت رشته میباشد.

 

جهت دسترسی به اسم function یا method ای که از طریق آدرس Url ارسال شده است طبق مثال زیر عمل کنید:

 $this->router->getMethod();

خروجی مثال بالا به صورت رشته میباشد.

 

جهت دسترسی به مقادیری که از طریق آدرس Url ارسال شده است طبق مثال زیر عمل کنید:

 $this->router->getRequests();

خروجی مثال بالا به صورت یک آرایه رشته ای از مقادیری میباشد که از طریق آدرس Url ارسال شده است و ترتیب این مقادیر مانند ترتیب مقادیری است که در Url وارد شده اند.
